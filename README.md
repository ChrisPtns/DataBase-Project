# 📚 University Library Management System (SQL)

Αυτό το project περιλαμβάνει τη σχεδίαση και την υλοποίηση μιας σχεσιακής βάσης δεδομένων για τη διαχείριση μιας πανεπιστημιακής βιβλιοθήκης. Περιλαμβάνει τη δημιουργία του σχήματος (schema), την εισαγωγή δεδομένων και μια σειρά από SQL ερωτήματα για την εξαγωγή χρήσιμων πληροφοριών.

## 🗂️ Δομή Βάσης Δεδομένων (Schema)

 Η βάση δεδομένων `university_library` αποτελείται από τους παρακάτω πίνακες[cite: 1, 11]:

*  **Authors**: Περιέχει τα στοιχεία των συγγραφέων (ID, όνομα, επώνυμο)[cite: 1].
*  **Publishers**: Στοιχεία εκδοτικών οίκων (Όνομα, Χώρα)[cite: 6].
*  **Books**: Πληροφορίες για τους τίτλους, το ISBN, το έτος έκδοσης και τη διαθεσιμότητα των αντιτύπων[cite: 2].
*  **Students**: Μητρώο φοιτητών με στοιχεία επικοινωνίας (Email, Τηλέφωνο) και το τμήμα τους[cite: 3].
*  **Loans**: Καταγραφή των δανεισμών, των ημερομηνιών λήξης και των επιστροφών[cite: 4].

## 📊 Ανάλυση Δεδομένων & Queries

* Το project περιλαμβάνει 15 έτοιμα SQL queries που καλύπτουν βασικές ανάγκες διαχείρισης[cite: 11]:

1.  **Ιστορικό Δανεισμών**: Αναζήτηση βιβλίων που έχει δανειστεί συγκεκριμένος φοιτητής μέσω του ΑΜ του.
2.  **Διαθεσιμότητα**: Εύρεση τίτλων που δεν έχουν δανειστεί ποτέ.
3.  **Παρακολούθηση Καθυστερήσεων**: Εντοπισμός φοιτητών που επέστρεψαν βιβλία μετά την ημερομηνία λήξης (Due Date).
4.  **Στατιστικά Στοιχεία**: Υπολογισμός συνολικών δανεισμών ανά φοιτητή και ανά βιβλίο.
5.  **Κατηγοριοποίηση Επιστροφών**: Χρήση της εντολής `CASE` για τον διαχωρισμό των επιστροφών σε "Εγκαίρως" ή "Με Καθυστέρηση".
6.  **Γεωγραφική Ανάλυση**: Λίστα βιβλίων από εκδοτικούς οίκους που εδρεύουν στην Ελλάδα.
